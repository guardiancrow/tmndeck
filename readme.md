# tmndeck - 対魔忍RPGのユニット管理アプリ

## はじめに

これは対魔忍RPGのユニットを管理するアプリケーションです

## 何ができる？

- ユニットの所持・不所持の管理
- 育成状態の確認
- 必要覚醒素材の把握
- 欲しいユニットが出る確率の把握

## 特長

- ブラウザだけで動作します
- Bootstrapを使用しレスポンシブでモダンなUIで表示します

## プロジェクトゴール

- 簡単に管理できるようにする
- 対魔忍RPGのプレイをより快適にする
- 推しキャラ愛をはぐくむ
